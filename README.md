# HR Employee Attrition Dashboard
This project analyzes employee attrition trends using Power BI visualizations.

## Tools Used
- Microsoft Power BI
- Microsoft Excel Dataset

## Features
- Interactive filters by Department and Age Group
- Attrition analysis using visual charts
- Department-wise employee distribution
- Hourly rate comparison by attrition

## Insights
- Most employees stay in the company (~88% No Attrition).
- Only about 12% of employees leave the company.
- Research & Development has the highest number of employees.
- Employees aged 30–40 form a major part of the workforce.
